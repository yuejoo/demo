## 1. Usage
All those three parameters are required.
```
Usage: ProductionLinearity -i=<inputFilePath> -o=<outputFilePath>
                           -p=<preBuildDays>
Production Linearity tool for helping to plan the production.
  -i, --InputFilePath=<inputFilePath>
         The input csv file.
  -o, --OutputFilePath=<outputFilePath>
         The input csv file.
  -p, --PreBuildDays=<preBuildDays>
         The number of pre-build days for production.
 ```
